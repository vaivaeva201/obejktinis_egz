# obejktinis_egz

Ši programa skirta analizuoti pateiktus tekstus. Ji suranda kokie žodžiai kartojasi įkeltame teksta ir kiek kartų kartojasi.
Sudaro cross reference lentelę, kurioje galime matyti kurioje teksto eilutėse yra pasikartojantys žodžiai.
Taip pat iš pateikto teksto gali surasti ir išvardinti visus URL adresus.

<h2>Naudojimosi instrukcija</h2>
<ol>
    <li>Susiinstaliuoti <b>make</b> (Windows naudotojams rekomenduojama per <a href="https://gnuwin32.sourceforge.net/packages/make.htm">GNUWin32</a>)</li>
    <li>Atsidaryti terminalą</li>
    <li>Klonuoti programos repozitoriją:
        <pre><code>git clone https://github.com/vaivaeva201/obejktinis_egz</code></pre>
    </li>
    <li><b>Užeiti į projekto aplanką:</b>
        <pre><code>cd objektinis_egz</code></pre>
    </li>
    <li>Paleisti norimą funkciją:
        <ul>
            <li><pre><code>make main</code></pre> - pagrindinė programa</li>
            <li><pre><code>make clean</code></pre> - išvalo programos sukurtus failus</li>
        </ul>
    </li>
</ol>
